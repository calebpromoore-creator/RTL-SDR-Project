# RTL-SDR-Project
I'm using a RTL-SDR usb dongle and my own dipole antenna to pickup radio stations, satellite weather images, and images from the International Space Station. The goal of this project is to be able to pickup weather images from satellites as well as images from the International Space Station. Once Im able to sucessfully able to pickup and record one of these broadcasts, I'll be able to translate these broadcasts into images. Once I've successfully done this, I can automate the process. I'll be posting images on this page.
